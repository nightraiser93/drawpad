# DRAW-00004 — Text tool on canvas

- **Slice value:** Covers the note-taking use case, not just freehand sketching — text and drawing live on the same surface.

## Intent
Add a text tool to the same Skia canvas used for drawing, so users can add notes alongside sketches rather than needing a separate feature/screen.

## Acceptance criteria
- [ ] User can add a text element to the canvas via a text tool (alongside pen/brush)
- [ ] Text element is editable (position, content) after placement
- [ ] Text is included in PNG export (DRAW-00001) and persisted stroke/element JSON (DRAW-00003)

## Subtasks
- [ ] Add text tool UI alongside pen/brush tool selector
- [ ] Implement text element placement + editing on Skia canvas
- [ ] Include text elements in PNG snapshot export
- [ ] Include text elements in persisted sketch JSON
