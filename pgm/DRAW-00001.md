# DRAW-00001 — Core drawing canvas

- **Slice value:** A usable sketching app on-device, even offline — draw and save a sketch, no networking required. Also the foundation every other slice builds on (stroke data model).

## Intent
Build the core sketching surface using react-native-skia so users can draw freehand strokes with a pen/brush tool and save the result. This is the base canvas + stroke data model that live viewing, persistence, and the text tool all depend on.

## Acceptance criteria
- [x] Expo + dev client project scaffolded and runnable
- [x] react-native-skia integrated (config plugin set up) as the drawing surface
- [x] User can draw freehand strokes with a pen/brush tool on the canvas
- [x] Sketch can be saved as a PNG to the device gallery (Skia snapshot → expo-media-library)

## Subtasks
- [x] Scaffold Expo + dev client project
- [x] Integrate react-native-skia + config plugin
- [x] Implement pen/brush drawing tool, capturing stroke points as a vector data model
- [x] Implement save-to-gallery via Skia snapshot + expo-media-library
