# DRAW-00003 — Persist and resume sketches

- **Slice value:** Sketches survive as editable work, not just flat exported images — user can close the app and pick a sketch back up later.

## Intent
Save sketch stroke data locally so sketches can be reopened and continued, not just viewed as a saved PNG. Stroke data (vector JSON) is persisted alongside the PNG export from DRAW-00001.

## Acceptance criteria
- [ ] Stroke data (vector JSON) persisted locally (AsyncStorage/SQLite) whenever a sketch is saved
- [ ] User can browse/select a previously saved sketch
- [ ] Reopening a saved sketch restores it on the canvas, editable (not just a flat image)

## Subtasks
- [ ] Design local storage schema for stroke JSON (AsyncStorage or SQLite)
- [ ] Save stroke data on sketch save, alongside PNG export
- [ ] Build a sketch list/picker screen
- [ ] Load stroke data back into the Skia canvas on reopen
